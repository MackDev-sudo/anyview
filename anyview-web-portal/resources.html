Our Resources

Github

We use GitHub for secure storing and updating of our site's codebase. It allows us to manage our projects efficiently with version control, collaborative features, and extensive integrations.

Design

Our website is designed using HTML and CSS. HTML provides the structure, while CSS ensures the site is visually appealing with responsive design principles, making our content accessible on any device.

Custom Scripts

We have custom scripts written in Python and JavaScript to handle various functionalities. We utilize libraries such as Paramiko for SSH connectivity, Requests for HTTP requests, Pandas for data manipulation, and CSV for handling CSV files.

API

We integrate with external APIs such as Pyfos and Brocade API to enhance our functionality. These APIs help us interact with external systems, enabling features like automated data fetching and remote system management.

Authentication

For authentication, we rely on Google Firebase. It provides secure, scalable, and reliable authentication services, allowing users to sign in with their Google accounts and ensuring their data is protected.

Database

Our data is stored in a remote SQL Database. This setup ensures data integrity, security, and accessibility, enabling us to handle large amounts of data efficiently and perform complex queries with ease.
